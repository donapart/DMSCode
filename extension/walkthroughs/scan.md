# Dokument scannen (OCR)

DMSCode kann Bilder und PDFs automatisch text erkennen (OCR).

### So funktioniert es:

1. **Importieren**: Kopieren Sie ein Bild oder PDF in Ihren Dokumentenordner oder nutzen Sie den Befehl `DMS: Dokumente importieren`.
2. **Scannen**: 
   - Klicken Sie mit der rechten Maustaste auf eine Datei im Explorer und wählen Sie `DMS: Dokument scannen`.
   - Oder nutzen Sie den Befehl `dms.scanDocument`.
3. **Ergebnis**:
   - Der erkannte Text wird als Metadaten zum Dokument gespeichert.
   - Sie können nun nach dem Inhalt suchen.

### Automatische Erkennung
Neu importierte Dateien werden automatisch in die OCR-Warteschlange eingereiht.
